# s\_link\_customer Table (337)

A connection between a customer and a link. Registered customers are identified when clicking on a link

## Fields

| Name | Description | Type | Null |
|------|-------------|------|:----:|
|id|Primary key|PK| |
|customer\_id|Connection to a customer that clicked on a link|FK [person](./person)| |
|link\_id|Connection to a link that the customer clicked on|FK [s_link](./s-link)| |
|shipment\_id|Which shipment the customer received.|FK [s_shipment](./s-shipment)| |
|pressed|When the link was clicked.|DateTime|&#x25CF;|
|link\_type|If this is a link to a link or an image (gif/jpeg). See SpmLink::LinkType|Enum [ShipmentLinkType](./enums/shipmentlinktype)|&#x25CF;|
|page|Indicates the page number for this gif link|Int| |
|http\_user\_agent|The HTTP_USER_AGENT reported when the customer opened this link|String(1024)|&#x25CF;|
