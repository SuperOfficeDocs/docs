# reply\_template\_attachment Table (278)

This table is used to connect attachments to reply templates (many-to-many).

## Fields

| Name | Description | Type | Null |
|------|-------------|------|:----:|
|id|The primary key (auto-incremented)|PK| |
|entry\_id|The id of the reply_template this entry is connected to.|FK [reply_template](./reply-template)| |
|attachment\_id|The id of the attachment this entry is connected to.|FK [attachment](./attachment)| |
