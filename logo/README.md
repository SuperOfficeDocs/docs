# Logos

Mintlify requires fixed filenames for logos. Do not rename these files.

| File | Mode | Color | Source file |
|------|------|-------|-------------|
| `light.svg` | Light mode (shown on light backgrounds) | Green `#06423e` | `SuperOffice_Green logo_RGB.svg` |
| `dark.svg` | Dark mode (shown on dark backgrounds) | Dune white `#f2efea` | `SuperOffice_Dune white logo_RGB.svg` |

To update, replace the file contents but keep the filenames.
